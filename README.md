# Student Course Registration System

## 📌 Project Description
This is a command-line Python application that helps manage students, courses, and course registrations for a training school. It allows an admin to add students and courses, register students into courses, and view registration details.

---

## 🚀 How to Run the Project

1. Make sure you have Python installed (Python 3+)
2. Open terminal in the project folder
3. Run:

```bash
python3 main.py
